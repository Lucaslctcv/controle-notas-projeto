## Requisitos Funcionais (exemplos)
1. O sistema deve permitir cadastrar o nome e a matrícula do aluno.
2. O sistema deve permitir inserir as notas das provas e trabalhos do aluno.
3. O sistema deve calcular a média final das notas do aluno automaticamente.
4. O sistema deve informar se o aluno está aprovado ou reprovado, com base na média.
5. O sistema deve listar todas as notas e médias dos alunos cadastrados.

## Requisitos Não Funcionais (exemplos)
- O sistema deve ser simples e fácil de usar, para professores sem experiência em informática.
- O sistema deve rodar em terminal (linha de comando), sem necessidade de interface gráfica.
- O sistema deve armazenar temporariamente os dados durante a execução.
